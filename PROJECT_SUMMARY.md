# ATLAS Memory Core 项目总结

## 🏔️ 项目概述

## 🏗️ 技术架构

### 核心模块
```
atlas-memory-core/
├── src/                    # 源代码
│   ├── core/              # 核心模块
│   │   ├── embedding.py   # 嵌入模型（已完成）
│   │   ├── storage.py     # 存储管理（已完成）
│   │   ├── retrieval.py   # 检索引擎（待开发）
```

### 技术栈
- **编程语言**: Python 3.12+
- **向量数据库**: Qdrant
- **嵌入模型**: nomic-embed-text-v1.5
- **依赖管理**: Poetry (通过pyproject.toml)
- **测试框架**: pytest
- **文档**: MkDocs + GitHub Pages
- **CI/CD**: GitHub Actions

## 🎯 核心创新

### 1. 零Token经济学
- 本地嵌入模型，避免云端LLM调用
- 智能指针系统，按需加载详细内容
- 本地压缩算法，减少上下文长度

### 2. 混合检索策略
- 向量相似度 + 元数据过滤
- 动态阈值控制
- 语义理解和结构化查询结合

### 3. 自优化闭环
- 基于艾宾浩斯遗忘曲线的评分衰减
- 使用频率加权的重要性评估
- 夜间自动优化和清理

### 4. 分层存储设计
- 热记忆：高频访问，内存缓存
- 温记忆：中等频率，向量数据库
- 冷记忆：低频访问，归档存储
